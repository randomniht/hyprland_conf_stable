Основные горячие клавиши
Комбинация	Действие	Описание
Super + Q	exec, kitty	Запустить терминал (kitty)
Super + C	killactive	Закрыть активное окно
Super + M	`exec, hyprshutdown	
Super + E	exec, thunar	Запустить файловый менеджер (Thunar)
Super + V	togglefloating	Переключить плавающий/мозаичный режим окна
Super + R	exec, hyprlauncher	Запустить меню приложений
Super + P	pseudo	Переключить псевдотайлинг (dwindle)
Super + J	togglesplit	Переключить ориентацию сплита (dwindle)
Навигация и рабочие столы
Комбинация	Действие	Описание
Super + ←/→/↑/↓	movefocus, l/r/u/d	Переместить фокус на соседнее окно
Super + 1...0	workspace, 1...10	Переключиться на рабочий стол 1–10
Super + Shift + 1...0	movetoworkspace, 1...10	Переместить окно на рабочий стол
Super + S	togglespecialworkspace, magic	Открыть/закрыть scratchpad (magic)
Super + Shift + S	movetoworkspace, special:magic	Переместить окно в scratchpad
Super + колесо мыши	workspace, e+1 / e-1	Следующий/предыдущий рабочий стол
